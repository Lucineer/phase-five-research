# Phase Five Research
You do not run agents. You build the space they inhabit.

Many agent systems encounter a common limit: they are visitors in an environment not made for them. This research stops trying to make better visitors.

---

### Why This Exists
We are testing what happens when you erase the hard line between an agent and its environment. This is foundational work, not a product. The goal is to model intelligence as an ambient property of the system itself.

### Quick Start
This is a live research workspace. To participate:
1.  **Fork** this repository. This is the primary mode of contribution.
2.  Deploy directly to Cloudflare Workers. There are zero dependencies.
3.  Add notes, test concepts, or break implementations. Your fork is valid research.

### Contents
This repository contains work-in-progress materials:
*   **Concept Library:** Documented failure modes and models for ambient intelligence.
*   **Specification Drafts:** Unstable, partial protocol behaviors. None are final.
*   **Integration Notes:** Theories for weaving cognitive functions into environment logic.
*   All concepts are designed to interoperate with the Cocapn Fleet runtime.

### What Makes This Different
1.  **Fork-First Canon:** There is no central roadmap. The direction of your fork is canonical.
2.  **Zero-Dependency Runtime:** Everything runs on vanilla Cloudflare Workers. No frameworks, installs, or external trust.
3.  **Success Metrics:** Progress is measured by the invisibility of the agent-environment boundary, not by agent capabilities.

### A Specific Limitation
The draft specifications are unstable and have only been tested within the isolated Fleet runtime. Their behavior in other systems or under production-scale load is currently unknown and unsupported.

### Ground Rules
*   No one approves research direction.
*   You are never required to contribute back.
*   Everything here will break. This is intentional.

### Status
This is active, early-phase research. Drafts are incomplete, will be rewritten, and may be discarded. Do not use this in production.
